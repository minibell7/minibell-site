# 블로그 글 쓰는 방법 (Sanity CMS)

이제 블로그 글은 **관리자 페이지(Studio)**에서 편하게 작성할 수 있습니다.

## 1. 관리자 페이지 접속하기

### 🏠 내 컴퓨터에서 쓸 때 (개발 중)
1.  터미널에서 `npm run dev`를 실행합니다.
2.  인터넷 창을 켜고 **[http://localhost:3000/studio](http://localhost:3000/studio)** 로 들어갑니다.

### 🌐 인터넷에 올린 후 (배포 후)
1.  내 사이트 주소 뒤에 `/studio`를 붙여서 들어갑니다.
    *   예: `https://minibell.com/studio`
2.  로그인 화면이 나오면 구글 아이디로 로그인합니다.

---

## 2. 글 작성 순서

1.  왼쪽 메뉴에서 **Post**를 클릭합니다.
2.  오른쪽 위의 **연필 아이콘 (Create new)**을 누릅니다.
3.  빈칸을 채워줍니다:
    *   **Title**: 글 제목을 적습니다.
    *   **Slug**: 옆에 있는 **Generate** 버튼을 누릅니다. (주소 생성)
    *   **Published at**: 클릭해서 **Current time**을 선택합니다. (예약 발행도 가능)
    *   **Main image**: (선택) 대표 이미지를 업로드합니다.
    *   **Body**: 본문 내용을 자유롭게 적습니다. (이미지 추가 가능)
4.  다 썼으면 오른쪽 아래(또는 위)의 초록색 **Publish** 버튼을 누르면 끝!

## 3. 글 수정/삭제

*   **수정**: 목록에서 글을 눌러 내용을 고치고 다시 **Publish**를 누르면 바로 반영됩니다.
*   **삭제**: 글 상세 화면에서 오른쪽 아래 **점 3개(...)** 버튼을 누르고 **Delete**를 선택합니다.
