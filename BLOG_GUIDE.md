# 📝 블로그 글쓰기 가이드

개발 지식이 없어도 괜찮습니다! 아래 순서대로 따라하면 누구나 쉽게 글을 쓸 수 있습니다.

## 1단계: 파일 만들기
1. VS Code 왼쪽 탐색기에서 **`posts`** 폴더를 찾으세요.
2. `posts` 폴더에 마우스 오른쪽 클릭 -> **"New File" (새 파일)** 클릭.
3. 파일 이름을 영어로 지어주세요. (띄어쓰기 대신 `-` 사용)
    *   예시: `my-first-diary.md`, `travel-to-seoul.md`
    *   **주의**: 뒤에 꼭 **`.md`**를 붙여야 합니다!

## 2단계: 필수 정보 입력하기 (머리말)
파일을 만들었으면, **맨 윗부분**에 아래 내용을 똑같이 복사해서 붙여넣으세요.
이 부분은 글의 제목과 날짜를 알려주는 중요한 부분입니다.

```yaml
---
title: '여기에 글 제목을 쓰세요'
date: '2025-11-27'
---
```

*   `title`: 작은 따옴표(`'`) 안에 원하는 한글/영어 제목을 쓰세요.
*   `date`: 오늘 날짜를 `연도-월-일` 형식으로 쓰세요.

## 3단계: 글 쓰기 (본문)
위의 `---` 밑에서부터 자유롭게 글을 쓰시면 됩니다.

### 꾸미기 팁
*   **제목**: `#`을 붙이고 한 칸 띄우면 큰 제목이 됩니다.
    *   `# 큰 제목`
    *   `## 중간 제목`
*   **강조**: 글자를 `**`로 감싸면 굵게 변합니다.
    *   `**이것은 굵은 글씨**`
*   **목록**: `-`를 붙이고 한 칸 띄우면 점이 생깁니다.
    *   `- 사과`
    *   `- 바나나`

## 4단계: 저장하고 확인하기
1. `Ctrl + S`를 눌러서 파일을 저장하세요.
2. 웹사이트의 **Blog** 메뉴에 들어가면 방금 쓴 글이 짠! 하고 나타납니다.

---
### 예시 파일 내용
```markdown
---
title: '오늘의 일기'
date: '2025-11-27'
---

# 오늘은 기분이 좋다
블로그를 처음 만들었다. 생각보다 쉽다!

## 앞으로 할 일
- 매일 글 쓰기
- 맛집 탐방하기
```
